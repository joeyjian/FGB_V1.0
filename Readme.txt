1 示例描述
    
    该示例程序主要描述了RF模块普通功能，提供了简单RF模块配置，通过该模块体现出rf定长数据接收功能。
    
    RX操作与效果：
    1）每次收到TX包会改变LED的状态。
    2）用demo板测试时2s未正确收到数据会亮红灯报错。
    3）通过串口打印发送完成标志。

    用户可以通过这个示例程序了解到RF模块的配置。用户可以以此示例进行RF模块的二次开发。
        

2 工程使用

2.1 硬件环境
    
    LED_R-------P06
    LED_B-------P05
    UART------- P13

    1）用跳线帽连接模块上的LED_R(P06)、LED_B(P05)，才能导通LED。
    2）通过串口连接P13，用串口助手查看数据.

    
2.2 软件环境
2.2.1 KEIL

    1）打开目录Project/Keil/Project.uvprojx，点击“Build All”，再点击“Start/Stop 
       Debug Session”下载程序，
    2）点击“Run”运行程序。
  
    
3 注意事项

    1）定长的数据接收需要注意和定长发送相配合，数据的接收长度寄存器值需配置和发送端一致，最好初始化直接一致。
    2）CE拉高才能接收数据。
